
Prerequisites:
1. Create a virtual environment using Pycharm
2. From https://github.com/ankit1234us/Udacity_bikeshare -
   Copy all files and folder in the same fodler which has the venv folder for the created Virtual Environemnt.
3. Copy the excel files for the 3 cities to be processed in the Static Folder
4. Check for dependencies in requirements.txt and install the missing dependencies

Run:
1. Run app.py
2. Open http://localhost:5000/ in Chrome Browser

Use the web app:
1. Select the inputs
2. Click on Find Bikeshare Data button
3. View results
4. Click on the Return button if want to get data for some other combination of inputs
